# 🧠 Otvet IQ — Образовательная AI-платформа

Современная платформа вопросов и ответов (Q&A), интегрированная с генеративным искусственным интеллектом Google Gemini. Проект позволяет пользователям задавать вопросы, обмениваться знаниями и получать мгновенные, контекстно-зависимые ответы от нейросети.

### 🔗 Стек технологий
* **Frontend:** React 18, React Router v6, кастомные хуки.
* **State Management:** Redux Toolkit (AsyncThunks, Slices).
* **Backend as a Service:** Firebase (Firestore Database, Authentication).
* **AI Integration:** Google Gemini API (модель gemini-2.5-pro).
* **Styling:** CSS Modules, Flexbox, Mobile-First подход.
* **Деплой:** Vercel (с настроенным роутингом).

### 🚀 Реализованный функционал
1. **AI с памятью контекста:** Нейросеть не просто отвечает на вопросы, но и "помнит" историю переписки в рамках каждого отдельного треда.
2. **Глобальный поиск и фильтрация:** Мгновенный поиск по всем вопросам через Redux, а также сортировка ленты (Новые, Популярные, Мои вопросы).
3. **Защита маршрутов (Route Protection):** Разделение прав доступа. Гости могут только читать ленту (срабатывают UI-ловушки с предложением регистрации), а создание вопросов и профиль доступны только авторизованным пользователям.
4. **Безопасная авторизация:** Вход через Google (Firebase Auth) с сохранением сессии.
5. **Real-time база данных:** Все вопросы, ответы ИИ и пользователей мгновенно сохраняются и синхронизируются в облаке Firestore.

### 💡 Особенности реализации
* **Продвинутый UI/UX:** Реализовано закрытие модальных окон и dropdown-меню по клику вне области (через хуки `useRef` + `useEffect`).
* **Безопасность:** Защита API ключей (Firebase и Gemini) через переменные окружения `.env`.
* **Оптимистичный UI:** Интерфейс реагирует мгновенно, не дожидаясь долгих ответов от сервера (используются лоадеры и блокировка полей ввода от спама).
* **Адаптивность:** 100% отзывчивая верстка под мобильные устройства, планшеты и десктопы.
